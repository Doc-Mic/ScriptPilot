# ScriptPilot

ScriptPilot is an AI-powered Android productivity app designed for YouTube creators. The app helps creators discover trends, generate ideas, create scripts, optimize SEO, and produce Shorts content using AI assistance.

---

## Features

- AI-powered YouTube script generation
- YouTube trend discovery
- Video idea generation
- Shorts content mode
- SEO assistant
- Premium dashboard UI
- Firebase Authentication
- Save and manage creator projects
- Modern Material 3 interface

---

## Tech Stack

- Kotlin
- XML
- MVVM Architecture
- Firebase Authentication
- Firestore
- REST APIs
- Material Design 3
